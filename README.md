# AI-LLM-project
🤖 Streamlit Gemini AI Assistant

A simple AI assistant built using Streamlit and Google Gemini API.
This app allows users to ask questions and receive AI-generated responses in real time.

🚀 Features

✅ Clean Streamlit interface

✅ Ask anything input box

✅ Gemini AI responses

✅ Button-triggered API call

✅ Error handling for empty input

✅ Uses free-tier friendly model

🛠️ Technologies Used

Python 3.11+

Streamlit

Google Generative AI (Gemini API)

📦 Installation
1️⃣ Clone the repository
git clone https://github.com/your-username/ai-assistant.git
cd ai-assistant
2️⃣ Install dependencies
pip install streamlit google-generativeai
🔑 Setup API Key

Go to: https://ai.google.dev/

Create a Gemini API Key

Replace in code:

client = genai.Client(api_key="YOUR_API_KEY")

⚠️ Never share your API key publicly

▶️ Run the App
streamlit run project.py

App will open in your browser.
